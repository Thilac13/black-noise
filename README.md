# 🎧 Black Noise

A **Progressive Web App (PWA)** that generates customizable black noise and ambient soundscapes to help you focus, relax, and sleep.

## Features

- 🔊 Black noise generation with adjustable parameters
- 📱 Install as a native app on any device (PWA)
- 🌙 Sleek dark-themed UI
- ⚡ Works offline
- 🎨 Smooth animations and modern design

## Live Demo

🌐 **[Open Black Noise](https://thila.github.io/black-noise/)** *(hosted on GitHub Pages)*

## Getting Started

No installation needed — just open the app in your browser and click play.

To install as a PWA:
1. Open the app in Chrome, Edge, or Safari
2. Click the **install** icon in the address bar (or "Add to Home Screen" on mobile)
3. Enjoy native-app experience!

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Web Audio API
- Service Workers (offline support)
- Web App Manifest (PWA)
